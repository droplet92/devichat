# devichat
![image](https://raw.githubusercontent.com/droplet92/devichat/master/2020-07-15-20-50-47.png)

simple chat server for Windows/Linux/FreeBSD

Usage
-----
server
```
$./rerun.tcsh [port_number]
```

Linux/Unix client
```
$cd client
$make
$./devichat [ip_address] [port_num]
```

Windows client
```
$cd winclient
$make
$./devichat [ip_address] [port_num]
```
